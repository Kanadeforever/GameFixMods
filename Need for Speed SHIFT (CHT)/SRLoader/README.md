# 原作者原文

---

这是一个可自由传播的免费软件，请勿通过任何途径购买。

## 关于本工具
这是一个因技术研究而诞生的工具，大约在三年前开发完成。它是针对securom 7、8序列号保护的通用工具，可以本地计算出securom的序列号以完成注册。

## 关于securom
securom是一个过时的保护软件，可以被欺骗注册、脱壳、虚拟机等方法轻易绕过，早已经被放弃开发。denuvo是继承自它的版本，强大得多。

## 关于发布
自工具开发完成以来，因当时大多国内游戏还在使用securom，故一直没发布。而在越来越多单机玩家愿意选择购买正版的今天，再由一个并不安全的保护系统来担任防盗版的“重任”，显然是有些可笑了，那就干脆让它结束吧。

---

上面都是废话，下面是一些使用说明：

1. 请先确认是由securom保护的，用一个叫“ProtectionID”的工具可查。
2. 使用方法：运行工具，点击右上角按钮，选择被securom保护的执行文件后，点击运行按钮。
3. 设置：
    - 如果运行后等待超过一分钟都没反应，工具会提示“注册失败”。
    - 出现上述情况，请尝试勾选“绕过游戏自校验”再试。
    - 如果依然失败，请尝试勾选“使用单核模式装载”。
    - 如果还是失败，请确认要运行的执行文件是否需要额外的命令行才能启动，如果是，请自行添加。
    - 如以上统统失败，那意味着无法使用。
4. 一些老版本的securom，在序列号保护之外还使用了光盘检测，可以在使用工具注册后，用虚拟光驱软件绕过光盘检测。
5. 本工具是执行本地注册，注册成功后不需要再使用工具运行。
6. 对于某些dlc，或注册后需要联网的，本工具无效，请不用尝试。
